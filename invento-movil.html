El telefono movil

Un teléfono móvil, también conocido como celular, es un dispositivo que se puede trasladar de un lugar a otro, puede establecer comunicacion sin que se registren inconvenientes en la llamada. El funcionamiento de este tipo de teléfono está dado por ondas de radios que le permiten acceder a las antenas que conforman la red de la telefonía móvil.

CARACTERISTICAS DE UN TELEFONO MOVIL

Es un dispositivo electrónico portátil que envía y recibe llamadas por medio de una portadora de radiofrecuencia. El dispositivo debe estar en el área de servicio para esto.

Pueden tener una gran cantidad de dispositivos integrados, como cámara, microfono, pantalla, entre otros.

FUNCIONES DE UN MOVIL

Principal y fundamentalmente, un telefono movil se empleada para enviar y recibir llamadas y mensajes de texto. Pero después se integraron los mensajes multimedia, radio, cámara, aplicaciones, juegos, y más. De esta forma fueron creciendo la cantidad de funciones que realiza este dispositivo.

NECESITA DE UN CARGADOR

Los teléfonos celulares funcionan a partir de una batería integrada o extraíble que se llena usando un cargador conectado a la corriente eléctrica.

Anteriormente se cargaban los celulares únicamente por cables, pero ahora hay otras formas disponibles en el mercado.
